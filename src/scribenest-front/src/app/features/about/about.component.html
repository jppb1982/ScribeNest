Quién soy

Soy desarrollador con 7 años de experiencia trabajando en distintos entornos y tecnologías. Esa trayectoria me dio versatilidad y una curva de aprendizaje rápida. Actualmente estoy profundizando en .NET 8 (C#) y Angular 16, con foco en arquitectura clara, código mantenible y buenas prácticas.

Sobre el proyecto

ScribeNest es un mini blog que armé para practicar y mostrar cómo estructuro una solución end-to-end. El backend está en .NET 8 con Entity Framework Core y SQLite, organizado en capas y aplicando Repository + Unit of Work. Incluye seed de datos para iniciar con contenido real.

El frontend está en Angular 16 (standalone) y consume /api/posts y /api/categories. Implementé búsqueda, paginación y filtro por categoría sincronizados por querystring, además de las páginas Home, Acerca de y 404. Mantengo la estética con Bootstrap 5 – tema Lux.

Qué aporta

  • Ejemplo funcional de .NET 8 + Angular 16 trabajando en conjunto.
  • Estructura por capas con EF Core + SQLite y patrones conocidos (Repository + UoW).
  • API simple y consumible; estado controlado por URL (búsqueda, filtros, paginación).
  • Buenas prácticas a nivel código: tipado estricto, componentes claros y servicios HTTP.
